# Silva Automation

The official business website for **Silva Automation LLC**, a specialized automation agency serving realtors and general contractors on Oʻahu.

## 🚀 Overview

This project is a modern, high-performance landing page built to convert visitors into automation clients. It features a "dark mode" professional aesthetic, GSAP-powered animations, and a fully responsive design.

**Live Site:** [silva-automation.com](https://silva-automation-git-main-savaiinaea-silvas-projects.vercel.app/)

## 🛠️ Tech Stack

* **Framework:** React 18 (via Vite)
* **Styling:** Tailwind CSS v4
* **Animations:** GSAP (GreenSock) & CSS Keyframes
* **Deployment:** Vercel
* **Package Manager:** NPM

## 🏃‍♂️ Getting Started

### Prerequisites
* Node.js (v18 or higher)
* NPM

### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/SavaiinaeaSIlva/silva-automation.git](https://github.com/SavaiinaeaSIlva/silva-automation.git)
    cd silva-automation
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

## 📂 Project Structure

* `/src/components/sections` - Main landing page sections (Hero, Problem, Solution)
* `/src/components/ui` - Reusable UI elements (Buttons, Cards, Scroll Indicators)
* `/src/styles` - Global CSS and Tailwind configurations
